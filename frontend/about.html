关于个人网盘

这是一个基于Go语言和Gin框架开发的前后端分离个人网盘项目。

技术栈

  • 后端：Go + Gin
  • 前端：HTML + CSS + JavaScript
  • 文件存储：本地文件系统
  • 系统监控：gopsutil

功能特点

  • 支持游客和管理员两种用户角色
  • 文件上传、下载、预览、重命名、移动和删除
  • 目录创建和管理
  • 多种文件类型的在线预览
  • 系统状态实时监控和历史趋势分析
  • 响应式设计，支持多种设备

使用说明

  • 游客可以查看文件详情和下载文件
  • 管理员可以上传、下载、重命名、移动和删除文件
  • 管理员账号：admin，密码：admin123
  • 文件上传大小限制：100MB
